Pengenalan Hosting

Apa Itu Hosting?

Hosting adalah layanan penyimpanan data digital di server yang selalu terhubung ke internet, sehingga website dapat diakses kapan pun dan dari mana pun. Server hosting berfungsi layaknya rumah bagi file website ‚Äî semua komponen seperti gambar, teks, video, database, dan file pemrograman disimpan di dalamnya. Setiap kali seseorang membuka website, browser akan mengambil data dari server hosting lalu menampilkannya di layar pengguna.

Cara kerja hosting: ketika pengguna mengetik alamat web di browser, browser akan menghubungi server tempat website disimpan, lalu menampilkan isi situs tersebut.

Jenis-jenis hosting:

  • Shared Hosting
  • VPS (Virtual Private Server)
  • Dedicated Server
  • Cloud Hosting

Pengenalan Control Panel

Apa Itu Control Panel?

Control Panel Hosting adalah antarmuka berbasis web yang digunakan untuk mengelola akun hosting dan semua sumber dayanya. Melalui control panel, pengguna dapat melakukan berbagai konfigurasi tanpa perlu mengetik perintah melalui terminal. Dengan control panel, pengelolaan server menjadi lebih mudah, cepat, dan aman, terutama bagi pengguna non-teknis.

Fungsi Control Panel:

  • Manajemen File

    • Mengupload, menghapus, atau mengedit file website melalui File Manager.

  • Manajemen Domain & Subdomain

    • Menambahkan domain baru, subdomain, dan mengatur DNS zone.

  • Email Hosting

    • Membuat akun email dengan domain sendiri (contoh: admin@namadomain.com ).

  • Database Management

    • Membuat dan mengelola database MySQL atau PostgreSQL.

  • Instalasi CMS Otomatis

    • Memasang WordPress, Joomla, atau Moodle hanya dengan beberapa klik.

  • Keamanan Server

    • Mengaktifkan SSL/TLS, proteksi DDoS, serta manajemen backup otomatis.

  • Monitoring dan Statistik

    • Melihat penggunaan CPU, RAM, storage, serta jumlah pengunjung website.

Contoh control panel populer: cPanel, Plesk, DirectAdmin, Webuzo.
